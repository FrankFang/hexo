本页面就是 Fly Label 的 Demo（IE 9 以下不用试了）。我比较喜欢第一个效果。

与其说这是一个 jQuery 插件，还不如说这是一个 Label CSS 动画合集。其原理就是通过改变 label 的 CSS 类名来实现相应的动画。如果你有不错的 CSS 动画代码，可以很容易地扩展它。

使用方法：

  1. 在页面中引入 jQuery 和 flyLabel.js
  2. 在页面中引入所有名字以 fly 开头的 CSS 类（从本页面的源代码中拷贝即可）
  3. 在脚本中添加$('xxx').flyLabels(); // xxx 是 Label 的父容器

我在手机上试了一下，效果也不错。
